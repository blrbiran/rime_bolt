## 声母表

| 全拼 | 双拼 | 示例 |
|:---:|:---:|:-----|
| zh | V | 中 yi**v**ong → zhong |
| ch | I | 吃 **i**i → chi |
| sh | U | 是 **u**i → shi |

其他声母保持不变：b p m f d t n l g k h j q x r y w z c s

## 常用组合速记

| 输入 | 输出 | 说明 |
|:---:|:----:|:-----|
| ui | shi | **u**=sh, **i**=i |
| vi | zhi | **v**=zh, **i**=i |
| ii | chi | **i**=ch, **i**=i |
| vs | zhong | **v**=zh, **s**=ong |
| is | chong | **i**=ch, **s**=ong |
| us | shong | - |
| vv | zhui | **v**=zh, **v**=ui |
| uk | shuai | **u**=sh, **k**=uai |

## 特殊功能键 (rime_ice)

| 按键 | 功能 |
|:---:|:-----|
| Shift | 切换中/英文 |
| Ctrl+Shift+4 | 切换简/繁体 |
| F4 | 方案选单 |
| \` | 辅码引导符 |
| uU + 拼音 | 部件拆字反查 |
| V + 符号 | 符号输入 |
